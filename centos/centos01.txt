Canales de ELRepo

Para organizar mejor el software que contiene el repositorio, el mismo se divide en 4 canales:

elrepo: canal principal del repositorio y único habilitado por defecto, contiene paquetes que no se encuentran en la distribución.
elrepo-extras: reemplazan otros presentes en la distribución.
elrepo-testing: reemplazan otros presentes en la distribución y 
elrepo-kernel: permiten actualizar el kernel actual a uno más reciente.

Para ver la lista de paquetes disponibles en un canal específico:

yum --disablerepo="*" --enablerepo="elrepo-testing" list available

yum --disablerepo="*" --enablerepo="remi" search ocs

net-tools:
Una colección de programas que forman el conjunto básico de herramientas de redes (NET-3) para el sistema operativo Linux.
La mayoría de ellos están obsoletos. Para el reemplazo compruebe el paquete de iproute. El paquete incluye: arp, hostname, ifconfig, netstat, rarp, route, plipconfig, slattach, mii-tool and iptunnel and ipmaddr.
URL         : http://sourceforge.net/projects/net-tools/

pciutils:
El paquete pciutils contiene varias utilidades para inspeccionar y configurar dispositivos conectados al bus PCI.
http://www.alcancelibre.org/staticpages/index.php/uso-lscpu-lsmod-lspci-lsusb

bind-utils:
Bind-utils contiene una colección de utilidades para consultar servidores de nombres de DNS (Sistema de nombres de dominio) para buscar información sobre servidores de Internet. Estas herramientas le proporcionarán las direcciones IP para nombres de host dados, así como otra información sobre dominios registrados y direcciones de red.
Debe instalar bind-utils si necesita obtener información de servidores de nombres DNS.



sudo yum install net-tools pciutils bind-utils -y

Optimizar y securizar



Optimizar disco:

Verificar:
cat /proc/mounts

ejemplo de particion root:
/dev/sda3 / ext4 rw,seclabel,relatime,data=ordered 0 0

sudo nano /etc/fstab

noatime,nodiratime


sudo mount -o remount,noatime,nodiratime,errors=remount-ro /

con lvs o ext4

sudo mount -o remount,noatime,nodiratime,defaults /

sudo mount -o remount,noatime,nodiratime,defaults /home

Verificar nuevamente:
cat /proc/mounts
/dev/sda3 / ext4 rw,seclabel,noatime,nodiratime,data=ordered 0 0
